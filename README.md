# Lego Harry Potter Years 5-7 — Archipelago Universal Tracker Pack
 
A tracker pack for **Lego Harry Potter Years 5-7** intended to be used with [Universal Tracker](https://https://github.com/FarisTheAncient/Archipelago). Logic is handled by the installed apworld made by jradcode23 of the game rather than being built into the pack itself.
 
## Installation & Usage
 
1. Download the latest version of the Universal Tracker apworld. Install it by dragging it onto the Archipelago Launcher or moving it into your custom_games folder. Also have the apworld for Lego Harry Potter Years 5-7 installed.
2. Download the latest release of this pack on the [releases](https://github.com/jonnytheagent/LHP5-7_AP_UniversalTracker/releases) page (`.zip`)
3. Place the `.zip` into a folder of your choice. Do **not** extract it, as that will not be compatible with Universal Tracker.
4. Open Universal Tracker in the Archipelago Launcher and connect to your server and slot. The tracker should prompt you with a window to choose the pack file. Once selected, you can use the "Map" tab in the tracker to see the locations.

## Frequently Asked Questions (FAQ)
* Why are there green squares for checks I can not do right now?

If there are available checks in for example areas that you have not progressed to yet, that means that you have met all the requirements to progress to that check with your current items. Common example: Just after starting out in the hub world, you will enter Kings Cross Station. The Gold Brick there will always be available with default settings in Sphere 1, since you always have access to a pet. To actually get that pet "spell", you have to progress through DADA Banned first and then backtrack.
